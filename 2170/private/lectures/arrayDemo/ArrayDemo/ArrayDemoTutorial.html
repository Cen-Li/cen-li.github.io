1. Enter array

Enter numbers in the box below the first header
all numbers must be seperated by comma(,)
For binary search, an ordered array is recommended but not necessary.

2. Show Array

Show Array operation is not necessary
but recommended before the implmentation of any algorithm.

3. Enter a number or location

For linear search, binary search or delete by number, entering a number in the number box is recommended, an alert will be triggered otherwise.
For delete by location, entering a location in the location box is recommended, an alert will be triggered otherwise.

4. Choose one Algorithm

Choose one algorithm before doing any operation.

5. Operation: Initial Order

Shows the intial state of array.
During 'Play', Play from the intial state of array again.

6. Operation: Previous Step

Shows the previous state of array.

7. Operation: Play

shows the whole procedure.

8. Opeartion: Next Step

Shows the next state of array.

9. Opearation: Final Order

shows the final state of array.

10. Operation: Speed Up

Speeds up the animation. You can click on it anytime.

11. Operation: Slow Down

Slows down the animation. You can click on it anytime.

12. Number box

Number in the box will be cleared in the begining of delete by location.
Number returned by algorithm will be added in the end of delete by location

13. Location box

Location in the box will be cleared in the begining of delete by number.
Location returned by algorithm will be added in the end of delete by number.
