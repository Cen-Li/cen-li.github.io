CSCI 2170
Peer Code Review Homework -- to be completed during Wednesday Closed lab period

In this lab, you will review OLA3B and OLA4A programs written by others in the class. There are two programs assigned for each of the OLA3B and OLA4A assignments.

This homework worth 100 points. Reviewing each program worths 25 points. Follow the steps below to review the programs:

  1. First, log into the ranger system;
  2. Start Firefox from within the ranger system, and log into PeerSpace;
  3. Click on "Tools", then "Peer Review";
  4. In this assignment, you are assigned 2 programs to review. Click on "Edit Review" to start reviewing each program.
  5. In the "Review Assignment" page:
    • Viewable Files: You can click on the file names of the "viewable files" to view their content. The file is displayed as an image. Click on the image to enlarge the file for viewing;
    • Downloadable Files: You can right click on the file name to download the file. Typically, the downloadable file are executable files or data files. Save them to a designated folder, for example "review".
      (You may use the Unix command: mkdir review, to create a new folder).
      You can then run the downloaded executable program in that folder.

      To make the "a.out" executable after downloading, do
      ranger% chmod u+x a.out

    • Assign points to individual items according to the program evaluation rubrics given. If you do not give full points to an item, you need to give a brief explanation in the space provided about why points are taken.
  6. In the "Feedback Received" page: After the other students in the class have finished reviewing your program, you will be able to view the assessments from your students.
