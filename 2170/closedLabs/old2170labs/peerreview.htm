Peer Code Review

Follow these steps:
  1. First, connect to and log into the ranger system,
  2. Launch the web browser from within the ranger system,
  3. Log into PeerSpace: peerspace.cs.mtsu.edu,
  4. Select "Tools" from the top menu bar, and from the drop down menu, select "Peer Review",
  5. You should see two programs written by students in the class waiting for review. Click on each program to review:
    • you may view the viewable files, i.e., the source file and log file. Click on the image to enlarge the fonts,
    • download the executable file to run the program
    • grade the program using the evaluation template provided
    • submit your review
  6. After others have reviewed your program, you may rate the reviewers according to the criteria provided.
