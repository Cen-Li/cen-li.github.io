In this lab, you will write two C++ programs to solve two problems. Two seperate projects need to be created, one for each problem. Create the second project after you have finished programming for the first problem.

Programming Problem One

Write a C++ program, named "average.cpp", that reads all the values from a data file, display the values and computes the average of these values.
