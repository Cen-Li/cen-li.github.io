Using MyProgrammingLab

We will be using the MyProgrammingLab (MPL) system that supports our textbook Starting out with C++ : From Control Structure through Objects (9th Edition) by Gaddis.

Step 1: Before you register

To use the MyProgrammingLab, you will need:    

  1. E-mail Address: use your MTSU mtmail account.
  2. Course (Section) ID: Given in class, and shown below. page.
  3. Access Code: either found inside the green card bundled with the textbook OR that you purchase online with a credit card (or paypal account).
IMPORTANT: Always use Firefox or Chrome for accessing MPL. Do NOT use Microsoft Internet Explorer (or Edge) for this process; IE does not work.

Step 2: Register

Visit www.MyProgrammingLab.com and click the "Student" button under "Register Now".

IMPORTANT: When you register, use the Course (Section) ID : PEAR-35876-SCAP-38
