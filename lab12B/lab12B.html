Lab 12B -- Programming C++ with Array Deletion

In this lab, we continue to practice performing Insert/Delete operations in one dimensional array. You will write a function that deletes an item from an array.

First create a new project, rename your "main.cpp" file to "deletebook.cpp". This program works as described below:

  1. First, it reads the titles of a collection of books from a data file and stores them in an array.
    The data file booktitles.dat can be copied using the command:

    cp ~cen/data/booktitles.dat workspace-directory/project-directory/.

  2. Once read, it prints out the list of book titles, one title per line.

    The above two steps have been completed in the program written for closed lab 11 and 12. You may copy/paste the part of the source code from those labs for these steps.

    Then, the program goes into a deletion loop. At each iteration of the loop:

    1. It prompts the user to enter a book title to be deleted from the array
      • If the book is present in the array, delete the book by shifting all the books after the specified book title one position up the array. Modify the size of the array appropriately.
      • If the book is not present in the array, give an error message to the user stating the fact that the book is not present.
    2. The loop terminates after the user press "enter" when prompted with a book title.

    Special Requirements

    • A user defined function named "DeleteByTitle" should be written for the delete operation. Think whether to use void or value returning function, what are the parameters for this function.
    • You are required to use the two functions we have studied: LinearSearch and DeleteByPosition for this program. These two functions are called from within the "DeleteByTitle" function.
