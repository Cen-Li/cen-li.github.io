Lab 5 -- Programming C++ with C++ type string and its functions

In this lab, we will practice: programming with C++ type string, its methods (built-in functions defined for C++ type string) and the getline function. Write a C++ program, named "count.cpp", that counts and reports the number of words in a line of text entered by the user. Assume the text only contains words separated by single space.

You are required to use C++ string functions for this problem.

Here are a few example runs of the program:

Sample run 1:

Please enter a line of text: The baby smiles as soon as she wakes up. There are 9 words in this sentence.

Sample run 2:

Please enter a line of text: Tom and Jerry are lovely characters. There are 6 words in this sentence.

Sample run 3:

Please enter a line of text: The Labor day weekend passed so quickly. There are 7 words in this sentence.

What to submit?

For this lab, submit the program count.cpp.
