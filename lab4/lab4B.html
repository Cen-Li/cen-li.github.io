Programming Problem

Write a C++ program named "coin.cpp" that simulates flipping a coin repeatedly and continues until three consecutive heads are tossed. At that point, your program should display the total number of coin flips that were made. Use rand() function to simulate random toss of the coin. Here are a few example runs of the program:

Sample run 2

Tail Head Tail Tail Tail Tail Head Head Tail Head Tail Head Head Tail Tail Head Head Head It took 18 flips to get 3 consecutive heads.

About Random Number Generation

  • How to generate random numbers in C++?
    • The function rand() produces a random integer number in the range between 0 and RAND_MAX (a constant defined to be the largest integer value). The function is defined in cstdlib header file, so include it in your program: #include <cstdlib>
  • How to make your program generate a different random sequence of toss each time you run the program? The function rand() generates pesudo random numbers based on a seed value. If we provide a different seed value each time we ran the program, then the sequence of random values generated on each run will be different. To do this, we are going to supply the seed value as the result of the function call "time(0)". The function call returns the number of seconds elapsed since Jan 1st, 1970. This number is going to be different each time we run the program, therefore a good random seed value to use. The function "time" is defined in a special header file, so include it in your program as: #include <ctime>
    To tell the program to use time(0) value of the seed, we supply this value as the parameter of the function srand(), (seeding for random number generator), like this: srand(time(0));
    It is important that we put this line as the first statement in the main function, right after the local variable declarations.
