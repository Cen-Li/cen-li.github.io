Lab 20 -- Sorted Linked List

In this lab, we will practice coding sorted linked list. We are not going to use a C++ class for this lab. The linked list will be developed entirely in the client program.

This program reads in a number of values from the data file and inserts the items into a sorted list in alphabetical order. The format of the data file is as the following:

apple pineapple orange milk ... Your program will read the items from the data file one at a time and inserts the item into the list. It repeats these read-insert steps til the end of the data file is reached. The content of the sorted list after each read-insert step is as following:
  • After the first item "apple" is read, the list content is head->apple.
  • After the item "pineapple" is read, the list is : head->apple->pineapple.
  • After the item "orange" is read, the list is : head->apple->orange->pineapple.
  • After the item "milk" is read, the list is : head->apple->milk->orange->pineapple.
