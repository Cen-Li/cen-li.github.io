Lab 10 -- Programming C++ using Linear Search and Selection Sort

In this lab, we will learn how to apply Search and Sort techniques learned in class to solve application problems.

  • First, create a new project. (This creates the project directory under the codelite workspace directory)
  • Then, copy the data file into the project directory. You can use 'cd' commands to navigate to the project directory, i.e., cd codelite/closedLab9. Now, copy the data file booktitles.dat with the command: cp ~cen/data/booktitles.dat .
Write a C++ program, named "books.cpp":
  1. The program reads the titles of a collection of books from a data file and stores them in an array.
  2. Then, the program sorts the book titles alphabetically in ascending order,
  3. Once sorted, the program prints out the list of book titles, one title per line,
  4. Afterwards, the program goes into a query mode prompting the user to query whether a book is in the collection. If the book is in the collection, it display its location in the collection. If the book is not in the collection, inform the user about that. The query mode ends when the usr enters an empty string, i.e., simply press the "ENTER" key when prompted for book title.

Notice: The index of the first element in the array is 0. Yet, for the output, the value corresponding to the first book should be 1. Same goes for the location values returned in the query mode.

The Search and Sort functions: int LinearSearch (const string a[], int aSize, string toFind) { // Look through all items, starting at the front. for (int i = 0; i < aSize; i++) { if (a[i] == toFind) return i; } // You've gone through the whole list without success. return -1; } void SelectionSort(string arr[], int n) { int i, j, minIndex; string tmp; // repeat pair-wise comparison across the elements n-1 times for (i = 0; i < n - 1; i++) { // find the index of the element with the smallest value in the remaining elements minIndex = i; for (j = i + 1; j < n; j++) { if (arr[j] < arr[minIndex]) minIndex = j; } if (minIndex != i) { // swap arr[i] and arr[minIndex] tmp = arr[i]; arr[i] = arr[minIndex]; arr[minIndex] = tmp; } } }
