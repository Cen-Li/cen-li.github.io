Lab 16 -- Unsorted List Class with Array Implementation

In this lab, we will practice adding methods to List class and using the List Class in a client program.

You can copy the current list class (list.h and list.cpp) into your account using the command:
cp ~cen/data/list.* workspace-folder/project-folder/.
(this command will copy both the list.h and list.cpp files into your directory). Copy/paste the list.h and list.cpp content into your current project.

Now, add the following 2 methods to the list class. Make sure to add the appropriate code in both the header file (list.h) and the specification file (list.cpp):

  • the copy constructor: this constructor will create a new object of the list class and initialize its values:
    • this list object will have the same values in its array as that of the object passedin as parameter
    • this list object will have the same array size value as that of the object passed in as parameter
  • AddAtFront method: add an item of the listItemType at the front of the list. The item to be added is supplied by the client program. If the list is already full, this operation can not be carried out. Display an error message in this case.
  • DeleteAll method: this method will delete ALL the occurrences of a value (of listItemType) in the list. The value to be deleted is supplied by the client program. Display a message indicating how many occurrences of the value were deleted.

To clear the end-of-file character ^d from the input stream, use the function call: cin.clear();
